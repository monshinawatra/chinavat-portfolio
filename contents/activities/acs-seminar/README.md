description | Showed how a team of AI agents can build software together — before multi-agent dev went mainstream.

Presented "Virtual Software Company with ChatDev" at the Applied Computer Science monthly seminar: how autonomous LLM agents can take on the roles of a real software team — product, engineering, testing — and collaborate to turn a single prompt into a working application. An early, hands-on look at the agentic-development paradigm that is now reshaping how software gets built.
