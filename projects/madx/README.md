name | MADX (MultiPlayer Ai Diffusion eXperience) is diffusion model trained on multi-agent environment. 
github | https://github.com/BetterACS/madx
description | MADX is a diffusion model designed for multi-agent environment.
(Currently supported only ATARI boxing environment). It is trained on a dataset of trajectories collected from multiple agents interacting in the environment. The model can be used as a ATARI boxing game. Which two players can play against each other inside the game.
